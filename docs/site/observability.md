Telemetry is optional and typically enabled with `team_id` and `project_id`.

## Tracks

- Session metadata
- Per-turn token usage
- Tool execution events and timings

## Dashboard

```bash
uv run rho-agent dashboard
```

Use this to inspect aggregate activity, usage, and tool behavior across runs.
